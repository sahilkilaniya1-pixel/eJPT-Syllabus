1. COMPUTER NETWORK & CORE TERMINOLOGIES
-------------------------------------------------------------------------
A Computer Network is a collection of computing devices that are connected 
in various ways in order to communicate and share resources.

🛠️ CONNECTION TYPES (Raste Ka Medium):
* Physical Wires/Cables: Ethernet cables, Fiber Optics, Coaxial cables.
* Wireless Connections: Radio waves, Wi-Fi, Cellular (4G/5G), Infrared signals.

🛠️ KEY TERMINOLOGIES:
* Node / Host:
  Network par jude kisi bhi physical device (Computer, Mobile, Printer, Router, 
  Server) ko generic term mein 'Node' ya 'Host' kaha jata hai.

* Data Transfer Rate (Bandwidth/Speed):
  Yeh woh speed hai jisse data network par ek jagah se doosri jagah move hota hai 
  (measured in Mbps / Gbps). Network performance aur optimization ke liye yeh 
  sabse key metric hota hai.

-------------------------------------------------------------------------
2. THE CLIENT / SERVER MODEL (Architecture Engine)
-------------------------------------------------------------------------
Modern computing ka sabse bada foundation Client-Server Model hai. Iska basic 
communication cycle aise kaam karta hai:

   [ Client ] ───( Sends Request )───> [ Server ]
   [ Client ] <───( Returns Response )─── [ Server ]

🛠️ SPECIALIZED SERVERS:
* File Server:
  Ek aisa dedicated computer jo network par multiple users ke liye files store, 
  manage, aur manage-permissions handle karta hai.

* Web Server:
  Ek aisa server computer jo web browsers (Client) se aane wali HTTP/HTTPS 
  requests ko process karta hai aur web pages (HTML, CSS, JS) return karta hai.

-------------------------------------------------------------------------
3. LOCAL-AREA NETWORK (LAN) & TOPOLOGIES
-------------------------------------------------------------------------
LAN ek aisa network hai jo ek chhote, local geographic area (jaise ek Office, 
Home, ya College Lab) ke machines ko aapas mein jotta hai.

🛠️ LAN TOPOLOGIES (Devices Ko Connect Karne Ke Visual Patterns):

1. Ring Topology:
   - Sabhi nodes ek closed loop (ring) mein jude hote hain.
   - Messages sirf EK HI DIRECTION (unidirectional) mein travel karte hain.
   - Issue: Agar ek bhi node beech mein crash hua, toh poora network down ho sakta hai.

2. Star Topology:
   - Isme ek Central Node (jaise Switch ya Hub) hota hai jisse baaki saare nodes jude hote hain.
   - Saare messages isi central node ke raste pass hote hain.
   - Advantage: Agar ek computer kharab hota hai, toh baaki network chalta rahega.

3. Bus Topology:
   - Sabhi nodes ek single main communication cable (Backbone Bus) se jude hote hain.
   - Messages us single line par DONO DIRECTIONS mein travel kar sakte hain.
   - Advantage: Setup karna bohot sasta aur simple hota hai.

-------------------------------------------------------------------------
4. GEOGRAPHICAL NETWORK CLASSIFICATIONS (LAN, MAN, WAN)
-------------------------------------------------------------------------
Network kitne bade ilaqe mein phaila hua hai, us hisab se iske Types hote hain:

* Local-Area Network (LAN):
  - Area: Single building, office, ya home.
  - Speed: Bohot fast data transfer rate.

* Metropolitan-Area Network (MAN):
  - Area: Ek poore shehar (City) ya bade university campus ka communication network.
  - Example: City cable TV network ya shehar ke banks ka private network.

* Wide-Area Network (WAN):
  - Area: Do ya do se zyada LANs ko bade geographic distance (Countries / Continents) 
    par connect karne wala network.
  - Gateway Node: LAN ka ek dedicated computer/router jo doosre networks se aane-jaane 
    wali saari communication ko handle karta hai.

🛠️ INTERNETWORKING & THE INTERNET:
ALag-alag networks ke beech mein communication ko 'Internetworking' kehte hain. 
Aur "The Internet" duniya ka sabse bada ultimate WAN hai jo poori globe ko aapas 
mein jotta hai.
