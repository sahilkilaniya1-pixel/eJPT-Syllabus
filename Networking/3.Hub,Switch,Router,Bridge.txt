1. HARDWARE DEVICES & OSI LAYER MAPPING
----------------------------------------------------------------------------------------------------

:: 1. REPEATER (Layer 1 - Physical Layer) ::
- Signal Amplifier/Regenerator: Ek electronic device jo weak huye input signals ko receive karta 
  hai aur unhe high power/level par retransmit (boost) karta hai.
- Purpose: Long distance cable length par signal degradation (attenuation) ko roknay ke liye 
  use hota hai.

:: 2. HUB (Layer 1 - Physical Layer - Non-Intelligent) ::
- Multi-port Repeater: Computers ko Star Topology mein jodta hai lekin isme koi intelligence nahi hoti.
- Working (Flooding Mechanism): Jab kisi ek port par packet aata hai, toh Hub use sabhi connected 
  ports par FLOOD (bhej) deta hai.
- Drawback: Bohot zyada network collisions (Packet Crashes) hote hain aur bandwidth waste hoti hai.

:: 3. MODEM (Modulator - Demodulator) ::
- Function: Computer ke digital signals (0s & 1s) ko analog sound signals mein badalta hai (Modulation) 
  taaki woh phone line par travel kar sake, aur receiving end par dubara analog se digital banata hai (Demodulation).
- Speed Metric: Data capacity ko Bits Per Second (bps) / Kbps / Mbps mein naapa jata hai.

:: 4. BRIDGE (Layer 2 - Data Link Layer) ::
- MAC-Based Filtering: OSI Layer 2 par kaam karta hai aur do network segments ko aapas mein jotta hai.
- Dynamic MAC Table: Communication ke dauran dynamically MAC addresses ka database banata hai aur 
  frames ko sirf target port par hi forward karta hai.

:: 5. SWITCH (Layer 2 - Data Link Layer - Multi-port Bridge) ::
- Intelligent Central Connector: Multi-port device jo Star Topology mein computers ko jotta hai.
- Working: Selective packet forwarding ke liye MAC Address Database (CAM Table) ka use karta hai. 
  Data flood karne ke bajaye sirf destination system ke dedicated port par hi bhejta hai.

:: 6. ROUTER (Layer 3 - Network Layer) ::
- Inter-Network Gateway & Path Selection: OSI Layer 3 par kaam karta hai aur alag-alag Local Area 
  Networks (LANs) ke beech data packets ko efficiently route karta hai.
- IP Addressing: Layer 3 Addressing (TCP/IP) ko samajhta hai.
- Broadcast Isolation: Yeh Broadcast Domain ko divide karta hai (ek network ke broadcast traffic ko 
  doosre network par nahi jaane deta).

:: 7. GATEWAY (All Layers / Layer 4 - Layer 7) ::
- Protocol Converter & Messenger Agent: Ek system se data lekar use interpret (translate) karta hai 
  aur doosre totally different protocol-based system ko transfer karta hai.
- Operating Level: OSI Model ki kisi bhi layer par operate kar sakta hai.

:: 8. WIRELESS ACCESS POINT - WAP / AP ::
- Wired-to-Wireless Bridge: Wireless devices (Mobiles, Laptops, Printers) ko Wi-Fi ya Bluetooth ke 
  zariye wireless network se jotta hai.
- Backbone Connection: Usually physical Ethernet cable se wired network se juda hota hai aur wired 
  va wireless devices ke beech data relay karta hai.
