1. CLASSIFICATION OF TRANSMISSION MEDIA
----------------------------------------------------------------------------------------------------
Data transfer ke liye signals do tarike ke mediums se pass hote hain:

A. Guided Media (Wired / Bounded):
   Physical cables ka use hota hai jahan signal ek physical raste se hokar jata hai.
   - Twisted-Pair Cable
   - Coaxial Cable
   - Fiber-Optic Cable

B. Unguided Media (Wireless / Unbounded):
   Bina kisi physical cable ke signals hawa mein (electromagnetic waves) travel karte hain.
   - Radio Waves
   - Microwaves
   - Infrared Signals

2. TWISTED PAIR CABLES & UTP CATEGORIES CHART
----------------------------------------------------------------------------------------------------
Twisted Pair Cable mein do insulated copper wires aapas mein twisted (lipti hui) hoti hain.
Iska main maqsad Electromagnetic Interference (EMI) aur external noise ko cancel out karna hota hai.

📊 UTP CATEGORIES CHART:

+--------------+--------------+-------------+--------------+-------------------------------------------------+
| UTP Category | Data Rate    | Max Length  | Cable Type   | Application / Common Uses                       |
+--------------+--------------+-------------+--------------+-------------------------------------------------+
| CAT1         | Up to 1Mbps  | -           | Twisted Pair | Old Telephone Cables                            |
| CAT2         | Up to 4Mbps  | -           | Twisted Pair | Token Ring Networks                             |
| CAT3         | Up to 10Mbps | 100m        | Twisted Pair | Token Ring & 10BASE-T Ethernet                  |
| CAT4         | Up to 16Mbps | 100m        | Twisted Pair | Token Ring Networks                             |
| CAT5         | Up to 100Mbps| 100m        | Twisted Pair | Ethernet, FastEthernet, Token Ring              |
| CAT5e        | Up to 1 Gbps | 100m        | Twisted Pair | Ethernet, FastEthernet, Gigabit Ethernet        |
| CAT6         | Up to 10Gbps | 100m        | Twisted Pair | GigabitEthernet, 10G Ethernet (55 meters max)   |
| CAT6a        | Up to 10Gbps | 100m        | Twisted Pair | GigabitEthernet, 10G Ethernet (55 meters)       |
| CAT7         | Up to 10Gbps | 100m        | Twisted Pair | GigabitEthernet, 10G Ethernet (100 meters full) |
+--------------+--------------+-------------+--------------+-------------------------------------------------+

🛠️ PROS & CONS OF TWISTED PAIR:
* Advantages (Fayde):
  - Cheap (Kafi sasta aur budget-friendly hota hai).
  - Easy to work with (Install aur handle karna bohot aasan hai).
* Disadvantages (Nuksan):
  - Low Data Rate (Coaxial aur Fiber ke mukable kam speed).
  - Short Range (100 meters ke baad signal weak ho jata hai).

3. COAXIAL CABLE (Copper Cable With Metal Shielding)
----------------------------------------------------------------------------------------------------
Coaxial cable ek khas type ki copper cable hai jisme signal interference ko rokne ke liye ek 
metal shield aur multi-layer insulation hoti hai. Yeh mainly Cable TV networks aur satellite 
antennas ko homes se connect karne ke liye use hoti hai.

🛠️ INTERNAL ANATOMY (Layers - Inside to Outside):
[ Inner Copper Conductor ] ➔ [ Insulator ] ➔ [ Outer Metallic Shield ] ➔ [ Plastic Outer Cover ]

🛠️ PROS & CONS OF COAXIAL CABLE:
* Advantages (Fayde):
  - Used in High-Frequency Applications (High frequency data carry kar sakta hai).
  - Improved Attenuation (Twisted pair se behtar signal stability aur shield effectiveness).
  - Less Susceptible to Noise (Noise aur external interference se kam prabhavit hota hai).
  - High Bandwidth & Reasonable Cost (Fiber se sasta, twisted pair se behtar bandwidth).
* Disadvantages (Nuksan):
  - Bulky, Thick & Stiff (Moti aur kadi hone ki wajah se modna mushkil hota hai).
  - Expensive to Install (Twisted pair ke mukable setup cost zyada hoti hai).
  - Security Issues & Grounding Necessary (Electrical grounding zaroori hoti hai).

----------------------------------------------------------------------------------------------------
4. FIBER OPTIC CABLE (Light Wave Data Transfer)
----------------------------------------------------------------------------------------------------
Fiber Optic cables digital data ko electrical signals ki jagah LIGHT PULSES (prakash ki kiran) 
ke roop mein saikdon miles tak ultra-high speed par carry karti hain.

🛠️ TWO MAIN TYPES OF FIBER OPTICS:
1. Single-Mode Fiber (SMF):
   - Choti core size hoti hai, ek baar mein ek hi light mode travel karti hai.
   - Long distance (saikdon kilometers) aur high speed ke liye best hai.
2. Multi-Mode Fiber (MMF):
   - Badi core size hoti hai, ek sath multiple light signals travel kar sakte hain.
   - Short distance (LANs / Data centers) ke liye use hoti hai.

🛠️ PROS & CONS OF FIBER OPTICS:
* Advantages (Fayde):
  - Extremely High Bandwidth (Duniya ki sabse fastest data transfer speed).
  - Thinner & Light-weighted (Patli aur halki hoti hai).
  - Zero Electromagnetic Interference (EMI Proof - Light par electrical interference ka asar nahi hota).
  - Longer Distance & Low Security Risk (Signal loss minimum hota hai aur data tap karna lagbhag asambhav hai).
* Disadvantages (Nuksan):
  - High Cost (Bohot mehngi hoti hai).
  - Very Fragile (Kanch/Glass ki bani hone ki wajah se jaldi toot sakti hai).
  - Difficult to Install (Special tools aur trained engineers ki zaroorat hoti hai).
  - Attenuation & Dispersion Issues (Lambi doori par light dispersion ko handle karna padta hai).
