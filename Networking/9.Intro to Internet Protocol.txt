🛠️ DETAILED BREAKDOWN OF PROTOCOLS BY LAYER (Hinglish Notes)
----------------------------------------------------------------------------------------------------

1. APPLICATION LAYER PROTOCOLS (User Facing & High-Level Services):
   - Telnet (Teletype Network): Remote terminal login ke liye text-based access setup karta hai.
   - FTP (File Transfer Protocol): Network par files ko send aur receive karne ke liye.
   - SMTP (Simple Mail Transfer Protocol): Electronic Emails (emails) bhejne ke liye standard.
   - DNS (Domain Name System): Domain names (e.g., google.com) ko IP Addresses mein convert karta hai.
   - RIP (Routing Information Protocol): Routers ke beech best routing path decide karne ka protocol.
   - SNMP (Simple Network Management Protocol): Network devices ko monitor aur manage karne ke liye.

2. TRANSPORT LAYER PROTOCOLS (Process-to-Process Delivery):
   - TCP (Transmission Control Protocol): 
     * Connection-Oriented protocol (Reliable, handshaking, error checking, retransmission).
   - UDP (User Datagram Protocol): 
     * Connection-Less protocol (Super fast, low latency, no delivery guarantee - useful for streaming).

3. NETWORK / INTERNET LAYER PROTOCOLS (Logical Addressing & Routing):
   - IP (Internet Protocol): Main protocol jo packets par Source & Destination IP address lagata hai (IPv4/IPv6).
   - ARP (Address Resolution Protocol): IP Address se Hardware MAC Address find karta hai.
   - ICMP (Internet Control Message Protocol): Error reporting aur diagnostic messages handle karta hai (e.g., Ping command).
   - IGMP (Internet Group Management Protocol): IP Multicasting (group transmission) ko manage karta hai.

4. DATA LINK & PHYSICAL LAYER STANDARDS (Hardware & Media Access):
   - Ethernet: Wired LAN connection ka sabse widely used standard (IEEE 802.3).
   - Token Ring: Legacy local area network technology jisme token passing mechanism use hota tha.
   - Frame Relay: High-speed WAN connection standard jo digital packet switching use karta hai.
   - ATM (Asynchronous Transfer Mode): Cell-based network technology jo voice, video, aur data fast transmission karti hai.

====================================================================================================
🌐 THE INTERNET PROTOCOL (IP) - CORE CONCEPTS & WORKING
====================================================================================================
Description: Network Layer ka sabse foundational protocol - Internet Protocol (IP), Packet Routing, 
aur Host Identification / Logical Addressing ka complete breakdown.

----------------------------------------------------------------------------------------------------
📄 SLIDE TEXT & DETAILED BREAKDOWN (Hinglish Notes)
----------------------------------------------------------------------------------------------------

1. PACKET-BASED DATA TRANSFER:
   - "Data is transferred in the form of packets via logical network paths in an ordered 
     format controlled by the network layer."
   - 💡 Explanation: Network Layer par bada data chhote-chhote **Packets** (Datagrams) mein 
     divide hota hai. Har packet ke sath Source IP aur Destination IP attached hota hai taaki 
     woh network par route ho sake.

2. ROUTING & PATH DETERMINATION:
   - "It does this by forwarding packets to network routers, which rely on algorithms to 
     determine the best paths for the data to travel. These paths are known as virtual circuits."
   - 💡 Explanation: Packets routers ke zariye aage pass hote hain. Routers Routing Algorithms 
     (jaise OSPF, BGP) ka use karke sabse best/shortest path chunne ka kaam karte hain.
   - 📌 Note for Exams/Interviews: Pure IP networks *Datagram-switching* follow karte hain 
     (jahan har packet independent path le sakta hai). Virtual Circuit concept connection-oriented 
     networks (jaise ATM / Frame Relay ya MPLS) mein use hota hai.

3. LOGICAL IDENTIFICATION (IP ADDRESSING):
   - "The Internet Protocol (IP) is the method or protocol by which data is sent from one 
     computer to another on the Internet. Each computer (known as a host) on the Internet 
     has at least one IP address that uniquely identifies it from all other computers on the Internet."
   - 💡 Explanation: IP Protocol internet par ek computer se doosre computer tak data bhejne ka 
     rule set hai. Internet se jude har system (Host) ko ek unique ID milti hai jise **IP Address** 
     kehte hain (e.g., IPv4: 192.168.1.1 ya IPv6).
